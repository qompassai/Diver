A hands-on Lua course built directly from the Diver Neovim configuration.

Each chapter teaches one Lua concept and then points at the real file in diver where that concept is used. Read the prose, then open the file and recognise the pattern. The reference modules on the left are runnable, fully-annotated Lua you can study line by line.
